# ООП - 2025
Проект был создан в рамках курса "Объектно-ориентированное программирование" от УрФУ студенткой группы КБ-201 Крамчаниновой Анастасией.
## Описание проекта
Телеграм и Дискорд боты для помощи с определением уровня языка и улучшением имеющихся знаний об языке.
## Основные функции, имеющиеся на данный момент
- Определение уровня языка в виде теста
- Тест на скорость
- Свой собственный словарь
- Сохранение данных в профиле
- Отложенные сообщения для: изучения, повторения и тестированием, с автодобавлением и изменением приоритетов слов в словаре

**Так же:**
- Все слова и тесты генерирует нейросеть Chat GPT5
