# OOP - 2025

This project was created as part of the "Object-Oriented Programming" course at Ural Federal University by Anastasia Kramchaninova, a student of group KB-201.

## Project Description

Telegram and Discord bots to help determine language proficiency levels and improve existing language knowledge.

## Main Features Currently Available

- Language level assessment through tests
- Speed test
- Personal vocabulary/dictionary
- Data saving in user profiles
- Scheduled messages for: learning, revision, and testing, with automatic addition and priority adjustment of words in the dictionary

**Additionally:**

- All words and tests are generated by the Chat GPT5 neural network
